Servicio de descargas

El objetivo de este servicio es el de permitir la descarga de algunas de las informaciones geográficas disponibles en www.geo.euskadi.eus.

  • Acceso por FTP
  • Acceso por HTTP

Puede accederse a este servicio de descargas directamente desde su navegador web (Chrome, Internet Explorer, Mozilla FireFox, etc.) o bien a través de programas específicos FTP que permiten optimizar la gestión de las descargas. Es el caso del software libre 'FileZilla Client' (http://filezilla-project.org/) que está desarrollado bajo los términos de la Licencia GNU.

Nota: El protocolo FTP, como medida de seguridad, puede estar restringido en organismos y empresas de gran entidad.

  • Acceso complementario para los datos LIDAR (sólo protocolo FTP): ftp://ftp.geo.euskadi.eus/lidar
